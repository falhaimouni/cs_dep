# CS Student Community

A frontend-only bilingual Arabic/English platform for a Computer Science student department/community.

## Stack

- React + Vite
- Tailwind CSS
- React Router
- i18next
- Framer Motion
- Netlify-ready static deployment

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Deployment

The app is ready for Netlify. Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```

Routing fallback is configured in `netlify.toml`.
